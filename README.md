# Amith C Kowshik — Portfolio Website

This is a simple responsive portfolio website built from your LaTeX resume. It includes sections for experience, projects, skills, education, awards, and contact links.

## Files

- `index.html` — main page content
- `styles.css` — responsive styling

## Preview locally

Open `index.html` directly in your browser, or serve the folder from a local web server.

### Optional local server

If you have Python installed:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This site is static and can be hosted on any static site provider.

### GitHub Pages

1. Initialize a Git repository in this folder.
2. Push to GitHub.
3. In repository settings, enable Pages from the `main` branch and select the root folder.

### Netlify or Vercel

Drag the project folder onto Netlify or link the GitHub repo from Vercel. Both services will deploy the static site automatically.
